노머니웨딩 개인정보 처리방침

최종 업데이트: 2026년 7월 5일

노머니웨딩은 결혼 준비를 돕기 위해 사용자가 입력한 정보를 기본적으로 기기 안(로컬)과, iCloud에 로그인한 경우 사용자 본인의 iCloud 계정에 저장합니다. 회원가입·로그인이 없고, 광고나 추적을 하지 않습니다. 다만 사용자가 ‘웹 링크로 공유’ 기능을 직접 선택하면 그 청첩장 내용이 공개 웹페이지로 발행됩니다(아래 웹 청첩장 공유 항목 참조).

저장되는 정보와 목적

다음 정보는 오직 앱 기능 제공을 위해 사용자의 기기와 본인 iCloud에만 저장됩니다.

  • 예식 정보 — 신랑·신부 이름, 예식 일시, 예식장 이름·주소 (청첩장·일정 표시용)
  • 하객 명단 — 하객 이름, 관계, 신랑/신부측, 참석 여부, 메모 (명단·참석 관리용)
  • 축의금 내역 — 받은 금액, 메모 (장부·통계·갚을 금액 계산용)
  • 청첩장 사진 — 사용자가 직접 선택한 커버·갤러리 사진 (청첩장 제작용)
  • 옵션 후기 — 사용자가 남긴 평가·지출 금액·한 줄 후기 (기기 안에 저장)

이 정보는 개발자에게 전송되지 않으며, Apple의 iCloud를 통한 동기화는 사용자 본인 계정 내에서만 이뤄집니다(Apple의 개인정보 처리방침이 적용됩니다).

권한 사용

  • 사진 — 청첩장에 넣을 사진을 사용자가 직접 고를 때만 접근하며, 선택한 사진 외에는 접근하지 않습니다. (허용 여부와 무관하게 사진 없이도 앱은 정상 동작합니다.)
  • 위치 — 기기의 위치(GPS) 권한을 사용하지 않습니다. 지도를 표시하기 위해 사용자가 입력·검색한 예식장 이름·주소 텍스트를 지도 서비스(Apple 지오코딩 및 카카오 로컬 검색)로 전송해 좌표로 변환하며, 검색어는 따로 저장하지 않고 사용자 신원과 연결하지 않습니다.

웹 청첩장 공유 (선택 기능)

청첩장을 만든 뒤 ‘웹 링크로 공유’를 선택하면, 받는 사람이 앱 설치 없이 볼 수 있도록 해당 청첩장 내용이 호스팅 서비스에 업로드되어 공개 링크(추측하기 어려운 고유 주소)로 제공됩니다. 이 기능을 사용하지 않으면 어떤 정보도 업로드되지 않습니다(일반 사용과 ‘이미지로 공유’는 업로드가 없습니다).

  • 업로드되는 내용 — 신랑·신부 및 혼주(부모님) 이름, 예식 일시·장소, 인사말, 연락처, 사용자가 입력한 계좌번호, 청첩장에 넣은 사진.
  • 업로드되지 않는 것 — 하객 명단, 축의금 내역, 옵션 후기(이 정보는 기기·본인 iCloud에만 남습니다).
  • 공개 링크를 아는 사람은 누구나 볼 수 있으므로, 외부에 알리고 싶지 않은 정보는 신중히 입력하세요.
  • 호스팅에는 데이터 저장(Supabase)과 페이지 표시(GitHub Pages)를 이용하며, 이는 사용자가 발행한 청첩장 표시 목적에만 쓰입니다.

제3자 제공 및 추적

노머니웨딩은 광고 추적을 하지 않으며, 제3자 분석·광고 네트워크에 어떤 정보도 제공하지 않습니다. 다만 지도·주소 검색 기능을 위해 입력한 주소 텍스트가 카카오 지도(로컬 검색) 서비스로 전송될 수 있으며(위 권한 사용 > 위치 참조), 이는 좌표 변환·지도 표시 목적에만 쓰이고 추적에 사용되지 않습니다. 웹 청첩장 공유 기능의 호스팅 이용은 위 웹 청첩장 공유 항목을 참고하세요.

아동의 개인정보

노머니웨딩은 아동을 대상으로 하지 않으며, 개발자가 개인정보를 수집하지 않습니다.
